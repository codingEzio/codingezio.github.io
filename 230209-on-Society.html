争论

愚昧

愚昧在很多时候并不是因为意见本身。
而是对其他意见不顾一切的敌视，排他。
不允许别人持不同意见，付诸行动对持不同意见者造成伤害，或是默认他人的类似行动。

单独个体如此，ta就是个愚昧的人。
环境如此，那就是环境本身充斥愚昧

大象

晚期互联网时代的言简地几乎所有能存在的公共议题都有一种刻意无视房间里的大象的默契，使得任何进行严肃讨论的尝试，尤其是关于责任群体划分的，都显得非常... 好笑。反正除了如此 YOLO (You Only Live Once) 三十年之外我是完全想不到任何能同时维持心理健康与财务健康的行为模式

战争

上一场战争中凋零的儿子，兄弟和爱人，有长眠在公园一样的墓地里，被世人爱与思念吗？名字有刻在石碑上，立在村口，被家乡铭记吗？

那些被战争伤残的年轻人，今天活的有尊严吗？被国家礼遇吗？老有所养吗？历史书里看得见他们的身影吗？媒体里听得见他们的声音吗？

只会感叹号，只会起大早在坟头蹦迪

雪崩

为什么雪崩的时候所有雪花一起落，为什么沙堆崩塌的时候所有沙粒一起滚，为什么你们上班下班睡觉起床时间都不约而同在一个区间里啊，一天三餐、上下班、用电高峰、流量使用时间集中，会带来多大的困扰和负担你知道吗，是不是有境外势力的指使有意约好的？

大清

今天站在历史的理解的角度，我们如果确知大清的时候，西方什么样子，可能会感慨，那时候落后的不是一点半点。但是过了40年，看今天我们和世界的差距，也许也会得出一个更客观的观点，也落后了不是一点半点。芯片，chatgpt等等，今天很多人可能还理解不了真正的差异在哪里。

「千年」文化沉淀下的「科学」

易中天与柳传志的对话

ref 我认为是烂在根上，千年来我们并不重视自然科学，顶层不需要，周边各国在这方面也没比我们好，中上层士大夫本身是文科屁股就是歪的。长期以来在管理层i 能重视而这些为为奇技淫巧。匠人更是低等存在，在技术领域的创新、著作都得不到应有的名和利。只是论为次等的存在。

从秦始皇带来的集权和统一之后，我们走向了与欧州不同的路线，这种集权在唐后就定形了。后面就在不停的重复 集权腐败 -> 底层压迫加剧 -> 崩坏革命 -> 诸强逐鹿 ->内战统一 -> 建国短暂繁荣。 根源逃不开集权对下面的予取予求，任何创新和私产都可以被轻易夺走。大家都去追逐权力。

ref 一缸水，如果没事淘洗大米，日积月累无意间还是能沉淀挺厚东西的。但男主人喝醉了砸一通，一群孩子抢东西撞个窟窿，这水换了一茬儿又一茬儿，可能最后只有砂子碎石子留在缸底吧。

ref 如果真要说有什么东西沉淀下来的话，那就是驭民之术了，不得不说还是很有效的，不然也不会在几千年的历史长河中不断重复循环，如果没有外力恐怕如今依然还在继续新的朝代变迁循环吧。

ref 自然/社会科学是同一套方法应用在不同研究对象上的结果：【现象归纳-演绎-理论建构-应用-证实/伪】循环。

这是目前科学研究唯一可行的路径，此方法下建构的理论是可解释/预测的、并在证实/伪中进步，不断扩大其解释力。

与其说是西方社会科学理论占统治地位，不如说是这套方法占统治地位。

ref 水利工程(运河、都江堰)、地动仪、浑天仪、节气、各种古建、中医、经文、诗作，能体现中国古人在地质学、天文学、气象学、建筑学、美学、医学、哲学、文学方面的建树，其中工程类学科的建树也间接证明古人在数学、物理学和植物学方面的造诣。但感觉大多是工程需求推动理论发展，少有理论界自主发展的。

ref 中国几千年沉淀下来的都是帝王之术，也就是说在中国科学要给政治让路，导致有时候科学并不能说真话，比如黄河流域的文物科学测定是公元前 1500 年前，但是为了营造中国有 5000 年的历史硬是说成是公元前 3000 年。还有疫情期间各种为了贴合政治需求的各种缺智力言论等
