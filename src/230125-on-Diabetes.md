## Overview

> Between *Type 1* and Type 2* Diabetes

### Comparison

| ATTRIBUTE for Type 1 | for Type 2 |
| :--- | :--- |
| Type 1 is the one covered by insurance | No |
| Type 1 might be treatable in the future | No |
| Type 1 is rare | More and more |
| Type 1 is not preventable | Able to mitigate |
| Type 1 is normally on people under 20 | After 45, statistically |
| Type 1 could be {environment, heredity} | keyword: Exercise, Obesity, Smoking |

### Similarity

- Not showing clear symptoms at the very start
- Not able to store and process *glucose* as expected
- Not treatable <small>(= back to normal, in the literal sense)</small>
- Not in a normal level of insulin that the body should have
- Could cause a whole bunch of worse problems like Hypertension<sup>Type 2</sup>
