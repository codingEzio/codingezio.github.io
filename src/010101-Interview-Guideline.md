## 问好问题

### The *Self*

| 分类 | 问题 |
| :---: | :--- |
| ***介绍*** | 个人技能\&经验 |
| ***优点*** | 可量化优点 |
| ***愿景*** | 研司领域与产品，己能带什么 |
| ***当前*** | 现在项目，热情\&输出 |
| ***规划*** | N年职规划，个技+公司层面 |
| ***项目*** | 宏观实现思路，栈力如设上线 |
| ***时代*** | 新技术观点\&做法 |
| ***协作*** | 调和与同事的异见 |
| ***调试*** | 调试bug的思路，预防与重构 |
| ***宏观*** | 项为己他的影响，忆新如*永测* |

### The *Interviewer*

- ***公司&产品***：内部文档库？产品上线流程？产品种类，竞争力，对标产品，下一步产品开发规划。
- ***工作流程&任务***：任务分配方式，项目周期，个人贡献评估，升职机制，管理方式，项目任务文档。
- ***技术栈&选型***：当前技术栈，选型理由，改进点，新技术看法，技术选型决策者。
- ***团队&个人发展***：上线若出问题如何应对，入职后任务，常接触的雇员，如何快速熟悉产品&开发流程，团队是否还在成长、规模，团队开发重点，人员强弱项，招人目的如为弥补还是强化。
- ***代码规范&测试***：代码规范，代码评审，测试团队。
