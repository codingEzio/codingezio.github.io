## GPL

- Keep the same license (i.e. GPL)
- Respect original *copyright notice*, *GPL text*, *author's attribution*
- Release the <small>(modified)</small> code fully
- Release the rights for anyone to modify and distribute it
- You can sell
- You can sell but you cannot take legal action if someone got a copy from illegitimate source
