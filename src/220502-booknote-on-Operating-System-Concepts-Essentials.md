### Notes

#### Definition

> An Operating System is
>
> - a software
> - a software that manages hardwares
> - a software that runs all the time
> - sometimes being called the 'Kernel'
> - in everywhere (=> every devices have an OS inside)

#### OS to the *User* (app <> OS <> hardware)

- Ease of Use
- Performance and Security
- Resource Utilization

#### OS to the *System*

- is a Resource Allocator
  - resources like CPU time, Memory/Storage space etc.
  - decide how to allocate RES to the USERs/PROGRAMs

- is a Control Program
  - control various I/O devices and user programs
  - manages exec of user programs (esp. I/O devices)
