### Step

> If you are using [*Intellij IDEA*](https://stackoverflow.com/a/71750515/6273859)

#### Text

1. Open *Preferences*
2. Find *Advanced Settings*
3. Check the `Allow auto-make to start even if developed ..` under `Compiler` tab
