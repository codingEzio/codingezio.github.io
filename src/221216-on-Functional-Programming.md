## Setup

> Of course, you can do functional programming without any of these tools mentioned here since it's more like a programming paradigm instead of language-specific.

### Installation

> Of course, I would assume you're on [*macOS*](https://en.wikipedia.org/wiki/MacOS_Monterey) and already had these installed
>
> - [`brew`](https://en.wikipedia.org/wiki/Homebrew_(package_manager)): the package manager
> - [*JDK*](https://codingezio.github.io/manage-jdk-version/)

#### Haskell

> References
>
> - [Installation - GHCup](https://www.haskell.org/ghcup/install/)

```bash
curl --proto '=https' --tlsv1.2 -sSf https://get-ghcup.haskell.org | sh
```

#### Clojure

> References
>
> - [Install Clojure](https://clojure.org/guides/install_clojure)

```bash
brew install clojure/tools/clojure


clojure --version
```

#### Elixir

> References
>
> - [Installing Elixir](https://elixir-lang.org/install.html#macos)

```bash
brew install elixir


elixir --version
```
