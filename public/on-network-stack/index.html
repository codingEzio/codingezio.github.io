Intro

Why

Back then, I remembered that I wondered about what exactly is the source code of CSS itself.

Eventually I came to a understanding that there were only specifications (like this one), and actual implementation were all up to the browsers (or browser engines like Blink which browsers like Chrome, Edge all depended upon, to be more specific) to implement.

What

I could, of course, read the thousands of RFCs on Internet Protocol Standards. Well, it's always good to read what the authors/implementers think or envision, but that stripped me from enjoying the fun of actually running, testing and maybe breaking them like I could do for a Vue.js front-end project.

In order to achive that, I have to find a way to implement them and getting the hella fun out of it 😁.

Current stage: finding good resources that could lead me to accomplish that.
