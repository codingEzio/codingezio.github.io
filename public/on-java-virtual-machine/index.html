More Details on JVM

The code you wrote would be firstly transformed into the .class file, then the JVM would execute that code which has all the 0s and 1s in it, like it was an actual machine running the machine code.

Since it behaves like an actual machine, in this very context, it would need to manage all the relevant stuff like memory management, garbage collection and of course, running the code and doing it efficiently.
