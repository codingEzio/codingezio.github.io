Optimization for Virtual Machine

Posted on August 03, 2022

Meta Info

  • Host: macOS Monterey (12.5)
  • Software: VMWare Fusion
  • Virtualized OS: Windows 10 (21H2) (OS build: 19044.1826)

Strategy

VMWare Tweaks
  • Tweak Display Resolution: as low as 1440 x 900 (crucial)
  • Tweak CPU Cores: >= 4 Process Cores
  • More RAM: 7200 MB Memory (half of those on your physical machine)
  • Disable 3D Acceleration
Inside the Virtual Machine
  1. Search Adjust the appearance and performance of Windows
  2. Select Adjust for best performance
