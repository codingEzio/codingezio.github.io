On Open Source Contribution

Posted on December 05, 2022

How

Git Pull Request

See more detailed or polished version written by others at here, here and here. Also this article was actually written on May 9, 2021, it was recently integrated into the Open Source Contribution blog post.

  1. Fork
  2. Clone your own fork
  3. Create a new branch for modifying: git checkout -b NEW_BRANCH
  4. Make changes (could be one or multiple)
  5. Push the changes to your own forked repository: git push origin NEW_BRANCH
  6. Click relevant buttons Compare & pull request on the Github
  7. Write relevant information like the title and more detailed description
