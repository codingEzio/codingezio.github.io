Installed Extensions

Reference to https://stackoverflow.com/a/49398449

Export
code --list-extensions > EXT_VSCODE_LIST.txt
Import (and install)
cat ~/EXT_VSCODE_LIST.txt
    | xargs -L 1 echo code --install-extension
    | % { "code --install-extension $_” }
