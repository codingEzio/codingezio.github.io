### Formatting and Linting

##### Java

> Reference to [https://stackoverflow.com/questions/50357628/google-intellij-java-style-changing-2-space-indentation-to-4-space-indentation](https://stackoverflow.com/questions/50357628/google-intellij-java-style-changing-2-space-indentation-to-4-space-indentation)

- 方案1
  - 在 Editor -> Code Style -> Java 调设置

- 方案2
  1. 下载 `google-java-format` 插件
  2. 重启 IDEA
  3. 在设置里将默认设置改为 *AOSP style*
  4. 需格式化时 按 Alt+Command+L
